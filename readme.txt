Para editar y ver el proyecto, cree una carpeta en su computador con el nombre "NumericalMethods" y descargue 
la carpeta "Calculadora" y "AN_environment". En esta, se encuentra el proyecto hecho con Django.

Una vez descargadas, abra la terminal en la carpeta NumericalMethods y ejecute el siguiente comando:

AN_environment\Scripts\activate

Esto activara un entorno virtual en cual está descargado Djando y las librerías necesarias para ejecutar la 
página.

Luego, acceda a la carpeta Calculadora ejecutando:

cd Calculadora

Y para correr el servidor, ejecute el siguiente coomando:

py manage.py runserver
